# Executive Financial Performance & Cost Analysis Dashboard

## Overview
This project is an executive-level financial dashboard inspired by Power BI-style reporting.
It provides leadership with a single view of revenue, costs, profitability, and margins to
support data-driven decision-making.

## Key Features
- Executive KPI cards (Revenue, COGS, Gross Profit, Net Profit, Margins)
- Operating and Net Profit trends over time
- Gross Margin analysis
- Operating expense (G&A) breakdown
- Executive summary with business insights

## Tech Stack
- Python
- Streamlit
- Pandas
- Plotly

## Use Case
Designed for finance leaders and business stakeholders to monitor financial performance
and identify profitability drivers.
